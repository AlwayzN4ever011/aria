Accessible Rich Internet Applications (WAI-ARIA) Editors' Drafts

This resource contains editors' drafts of specifications and related documents for Accessible Rich Internet Applications (WAI-ARIA). This technology is developed by the World Wide Web Consortium (W3C) Protocols and Formats Working Group (PFWG), with some modules jointly developed with other groups.

Editors' drafts have no official standing and may not reflect consensus of the Working Group. They are provided her to help stakeholders follow the latest work. The above editors' drafts are snapshots of the sources maintained in the W3C ARIA GitHub repository.

Currently available editors' drafts:

  • Accessible Rich Internet Applications (WAI-ARIA) 1.1 Editors' Draft
  • Core Accessibility API Mappings (Core-AAM) 1.1 Editors' Draft
  • Accessible Name and Description: Computation and API Mappings (AccName-AAM) 1.1 Editors' Draft
  • HTML Accessibility API Mappings (HTML-AAM) 1.0 Editors' Draft
  • SVG Accessibility API Mappings (SVG-AAM) 1.0 Editors' Draft
  • WAI-ARIA 1.1 Authoring Practices Editors' Draft

Official versions of the above specifications are published on the W3C site as Technical Reports. Currently available official publications:
